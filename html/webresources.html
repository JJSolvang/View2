  1. W3schools
    W3schools is an incredible online library of programming videos and tutorials. Visitors can access hundreds of modifiable programming lessons for free.

    • HTML
    • CSS
    • JavaScript
    I have used this resource for more than five years now, and I will continue to use it as a reference for many more to come.
    Rating: 5/5

  2. Code Cademy
    Yet another online library of programming videos and tutorials, this one geared more towards learning quickly and efficiently in large quantities.

    • HTML
    • CSS
    • JavaScript
    I have heard nothing but good things about this resource and I look forward to using it further in the future.
    Rating: 5/5

  3. W3C Markup Validation Service
    W3C is a great resource for verifying the integrity of your webpage HTML.
    I have used this resource in previous classes and and recommend it as a must have in the process of finalizing your webpages for launch.
    Rating: 4/5

  4. YouTube
    YouTube is amazing for listening to software developers discuss the programming process, problem solving, and what programmers wish they did when starting their programming journeys.

    • Software Problem Solving and Learning
    I often surf YouTube to learn about programming techniques and stay inspired.
    Rating: 4/5

  5. Khan Academy
    The last online library of programming videos and tutorials on this list, this one geared more towards younger audiences.

    • HTML and CSS
    • JavaScript
    • SQL
    I took these tutorials when I was in middle school and they didn't do a great job of setting me up for success. My biggest issue now is them introducing JavaScript before HTML and CSS, however, I can see their value, now.
    Rating: 2/5
